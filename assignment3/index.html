The HTML <picture> Element

The html <picture> element gives developers more flexibility in displaying images. It contains one or more <source> elements and the <srcset> attribute is used to refer to different images allowing the browser to use the image that best fits the view or device. Some browsers might not support all image types, by using different image formats the browser will select the first one it recogizes and ignore the rest. Each <source> has a <media> attribute that defines when each image is most suitable. The <img> element is used for browsers that don't support the <picture> element or if the <source> tags don't match.

The <SVG> Element

SVG stands for Scalable Vector Graphics, <SVG> is a container element for these types of graphics. It has multiple methods for drawing shapes, logos, text, etc. <SVG> graphics can be modified with any text editor and don't lose any quality if zoomed or resized. You can also embed <SVG> images directly into your html code.

SVG Code explanation:

  • An SVG image begins with an <SVG> element
  • The width and height attributes of the <SVG> element define the width and height of the SVG image
  • The <circle> element is used to draw a circle
  • The cx and cy attributes define the x and y coordinates of the center of the circle. If cx and cy are not set, the circle's center is set to (0, 0)
  • The r attribute defines the radius of the circle
  • The stroke and stroke-width attributes control how the outline of a shape appears. We set the outline of the circle to a 4px purple "border"
  • The fill attribute refers to the color inside the circle. We set the fill color to pink.
  • The closing <SVG> tag closes the SVG image

The <select> Element

The <select> element represents a control that provides a menu of drop-down options. It is <form> element and is used with a <label> element which contains text to introduce the select element. This is used with one attribute, 'for', whose value should match the id of the associated <select> element. The <select> element's opening tag will contain 2 attributes. First, the 'name' attribute will name the dropdown element, and second, the 'id' attribute which pairs the <select> element with the <label> element. Next we put an <option> tag for each list item, and use a 'value' attribute that specifies the value sent when the form is submitted.
