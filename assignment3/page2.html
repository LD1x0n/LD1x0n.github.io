CSS Background-Image Property

The background-image property is used to set a background image for an element. By default the image will repeat and is placed in the top-left corner of the element. This can be changed using the background-repeat property. A URL value is used to specify the image or multiple images can be used by seperating them with a comma.

Source www.w3cschools.com

The Text-Shadow Property

The text-shadow property adds a nice shadow to your text. The syntax of the property is text-shadow: (offset-x offset-y blur-radius color). Each shadow effect must specify a shadow offset and may optionally specify a blur radius and shadow color.

Source www.w3cschools.com

The Cursor Property

The cursor property is used to make custom cursors. To do this you have to tell CSS what image to use using the URL value, then apply it to the element you want it to appear on when you hover over it, like the body or a div element. The next property is the fallback in case your image doesn't load so you won't be cursorless. The cursor property accepts SVG files, no animated gifs though.

Source freecodecamp.org
